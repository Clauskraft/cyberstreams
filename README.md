**One-command platform initialization** that automatically:

- ‚úÖ **Starts Ollama service** (port 11434)
- ‚úÖ **Installs AI models**:
  - `dolphin-llama3:8b` - Uncensored OSINT analysis (4.7GB)

  - API server (port 3001)

- ‚úÖ **Loads OSINT startkit** with 50+ intelligence sources

### üîÑ Real-time Data Ingestion

#### **`scripts/cron/ingest.ts`**

**Automated RSS ingestion pipeline** that:

- ‚úÖ **Parses RSS feeds** from authorized intelligence sources
- ‚úÖ **Stores in PostgreSQL** with vector embeddings for semantic search
- ‚úÖ **Distributes to MISP** (Malware Information Sharing Platform)
- ‚úÖ **Publishes to OpenCTI** (Open Cyber Threat Intelligence)
- ‚úÖ **Generates STIX 2.1** indicators for threat intelligence sharing
- ‚úÖ **Runs as cron job** for continuous data updates

```bash

npx ts-node scripts/cron/ingest.ts

# Or configure as cron job for hourly updates
# Add to crontab: 0 * * * * cd /path/to/cyberstreams && npm run ingest
```
